# Diabetes Diagnosis

## Giới thiệu
Ứng dụng **Diabetes Diagnosis** là ứng dụng sử dụng công nghệ VR tích hợp AI 
nhằm giúp người dùng có những trải nghiệm trực quan nhất về trường Đại Học Bách Khoa Hà Nội.
